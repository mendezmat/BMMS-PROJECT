# BMMS Hotfix 023 — Authoritative Verse Size

The fitting engine previously replaced the operator-selected size.

This build changes that:
- `appearance.bodySize` is the final rendered font size.
- Smart Balance only distributes words across lines.
- BMMS adds lines instead of shrinking the text.
- TV Broadcast panels may grow vertically.
- The default verse size for new state is 72 px.
- ProPresenter, Manual, Preview, Program, Take, Clear and Auto routes
  remain unchanged.
