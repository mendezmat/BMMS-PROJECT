# SPEC-004 — BMMS Visual Editor Core

## Milestone 0.4.0

The initial editor must provide:

- 1920×1080 stage;
- selectable elements;
- direct drag and resize;
- text and shape creation;
- layer selection and ordering;
- contextual property inspector;
- document background;
- undo and redo;
- persistent document storage;
- keyboard save and undo shortcuts.

## Next milestones

- image elements and file upload;
- rotation handle;
- alignment guides and snapping;
- multi-selection;
- templates and module bindings;
- separate preview/program output;
- animation timeline;
- responsive safe areas.
