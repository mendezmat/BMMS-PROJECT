# SPEC-004 — Lower Third

## Purpose

Lower Third creates reusable name, role, title, speaker, and informational graphics for live production.

## Core fields

- Primary line.
- Secondary line.
- Optional tertiary line.
- Optional icon or logo.
- Optional category.
- Optional accent label.

## Functional requirements

- Real-time WYSIWYG editor.
- Reusable presets.
- Template library.
- Dynamic text fields.
- In/out animation.
- Preview/program separation.
- Multiple saved lower thirds.
- Keyboard and MIDI trigger support in future versions.
- Platform-independent overlay URL.

## Modes

- Person identification.
- Speaker identification.
- Song or segment title.
- General information.
- Custom text.

## Acceptance criteria

- A lower third can be created in under one minute.
- Text changes update preview immediately.
- Templates can be duplicated and saved.
- Live output can be taken in and out reliably.
- Long names produce warnings or adaptive layout behavior.
