# Release Readiness

## Alpha

- Core workflows function.
- Known data-loss defects are absent.
- Manual recovery exists.
- Internal testing only.

## Beta

- Main use cases are complete.
- Installer or packaging is available.
- Error reporting is usable.
- Documentation covers setup and operation.
- Benchmark exists for Smart Engine features.

## Release Candidate

- No known critical defects.
- Migration paths tested.
- Overlay compatibility tested in target platforms.
- Performance targets measured.
- Changelog and release notes complete.

## Stable

- Production use validated.
- Support policy defined.
- Backward compatibility policy active.
- Recovery and backup procedures documented.
