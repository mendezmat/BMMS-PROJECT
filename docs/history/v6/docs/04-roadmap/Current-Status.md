# Current Status

## Confirmed project direction

- BMMS is a modular platform.
- BMMS Graphics is one application within the platform.
- BMMS TB Tool is a separate application in the same family.
- Foundation is shared infrastructure.
- Smart Engine is independent and reusable.
- Overlays use browser-compatible URLs.
- vMix and OBS are target platforms.
- ProPresenter integration is part of Scripture.
- Editing should be visual and real time.
- Smart Engine assists the operator and does not publish autonomously.
