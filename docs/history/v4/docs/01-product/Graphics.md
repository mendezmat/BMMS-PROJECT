# BMMS Graphics

## Objetivo

Aplicación dedicada a la generación de gráficos para producción en vivo.

## Módulos

### Scripture
Gráficos bíblicos conectados a ProPresenter.

### Smart Flyer
Transformación automática de flyers en gráficos broadcast.

### Lower Third
Creación rápida de identificadores y títulos.

### Countdown
Temporizadores reutilizables.

## Diseño

Todos los módulos comparten:
- Foundation
- Smart Engine
- Sistema de Overlays
- Editor visual en tiempo real
