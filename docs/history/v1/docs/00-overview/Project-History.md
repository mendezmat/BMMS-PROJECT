# Historia del Proyecto

## Evolución conocida

1. PPVerse Bridge
2. vMix Graphics
3. vMix Graphics Studio
4. BMMS Graphics
5. BMMS Platform

## Evolución técnica

El proyecto comenzó como una herramienta específica para gráficos de vMix y evolucionó hacia una plataforma modular. Durante esa evolución aparecieron módulos especializados como Scripture, Lower Third, Smart Flyer y Countdown, además del concepto de Smart Engine como motor compartido.

Posteriormente se definió que BMMS debía soportar múltiples plataformas (vMix, OBS y futuras) sin modificar el núcleo de la aplicación.
