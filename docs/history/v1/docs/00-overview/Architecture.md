## Foundation

Servicios compartidos por toda la plataforma.

## Smart Engine

Motor inteligente reutilizable para OCR, análisis semántico, composición automática y futuras funciones.

## BMMS Graphics

Aplicación enfocada en overlays y gráficos para producción.

## BMMS TB Tool

Aplicación destinada al monitoreo visual de Talkback.
