# BMMS 0.8.0 — Editor Experience

Implemented:

- multi-selection with Shift/Ctrl click;
- marquee selection on the canvas;
- multi-element dragging;
- copy, paste and duplicate;
- grouping and ungrouping;
- alignment and distribution;
- keyboard nudging by 1 or 10 pixels;
- select all;
- layer visibility and locking;
- inline layer renaming;
- drag-and-drop layer ordering;
- multi-selection inspector;
- expanded editor keyboard shortcuts.

## Shortcuts

- `Ctrl+C`, `Ctrl+V`, `Ctrl+D`
- `Ctrl+G`
- `Ctrl+A`
- arrow keys and `Shift + arrow`
- `Delete`
- `Ctrl+Z`, `Ctrl+Shift+Z`
