# BMMS 0.4.0 — Visual Editor Core

Implemented:

- shared graphics document model;
- element creation and normalization;
- immutable element operations;
- bounded undo/redo history;
- visual editor route;
- 1920×1080 broadcast stage;
- text and shape tools;
- drag and resize;
- contextual inspector;
- layers panel and ordering;
- persistent document endpoint;
- editor tests.

Open the editor at:

`http://localhost:4173/editor`
