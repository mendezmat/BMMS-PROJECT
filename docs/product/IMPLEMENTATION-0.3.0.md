# BMMS 0.3.0 — Visible ProPresenter and Scripture configuration

## Implemented

- navigation between Lower Third, Scripture and Integrations;
- global ProPresenter configuration form;
- host/IP, port, protocol, auto-connect and reconnect controls;
- monitoring controls for Scripture and media;
- persistent global application state;
- Scripture source selector: Manual or ProPresenter;
- organized Scripture sections;
- shared contextual appearance editing model;
- live Scripture composition preview;
- simulated ProPresenter Scripture events;
- Server-Sent Events for incoming application updates;
- validation and tests.

## Deliberately deferred

The production ProPresenter transport remains deferred until the supported
ProPresenter version and API contract are selected and tested against a real
installation.
