# BMMS 0.7.0 — Broadcast Components

Implemented:

- component library drawer;
- Scripture component factory;
- Lower Third component factory;
- editable component-generated layers;
- component metadata;
- component group selection and movement;
- component deletion;
- transparent output option;
- output renderer transparency support;
- component API;
- component factory tests.

## Operator workflow

1. Open the Visual Editor.
2. Select **Componentes**.
3. Insert **Versículo** or **Lower Third**.
4. Edit any generated layer normally.
5. Hold `Alt` while selecting or dragging to operate on the full component.
6. Enable **Salida transparente** for use in vMix or OBS.
