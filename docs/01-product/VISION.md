# Visión del producto

BMMS no es una aplicación única. Es una plataforma modular para resolver problemas técnicos reales en producción audiovisual, broadcast, iglesias y eventos.

## Propósito

Reducir trabajo manual, mejorar consistencia visual, acelerar la operación técnica y permitir que distintas herramientas compartan una misma base.

## Principios

- Modularidad
- Escalabilidad
- Bajo acoplamiento
- Interfaz coherente
- Automatización útil
- Compatibilidad con múltiples plataformas
- Independencia de proveedores
- Validación con datos reales

## Experiencia objetivo de Smart Flyer

1. Arrastrar un flyer.
2. Analizarlo automáticamente.
3. Detectar título, fecha, hora, lugar, referencia y QR.
4. Generar un CG.
5. Ajustarlo visualmente.
6. Publicarlo por la salida correspondiente.
