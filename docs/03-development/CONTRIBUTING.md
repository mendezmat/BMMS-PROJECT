# Contribución

## Ramas sugeridas

- `main`: estable
- `develop`: integración
- `feature/*`: funciones
- `fix/*`: correcciones
- `docs/*`: documentación
- `release/*`: preparación de versiones

## Commits

```text
tipo(alcance): descripción
```

Ejemplos:

```text
feat(smart-engine): add OCR provider interface
fix(scripture): prevent overflow in long verses
docs(architecture): document overlay URLs
```
