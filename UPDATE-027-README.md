# BMMS Update 027 — Editor visual del modo simple

## Cambios

- Los controles del Preview ahora están en una barra horizontal superior.
- Área segura, cuadrícula y Snap configurables.
- Selector visual de la composición sobre el Preview.
- Arrastre para modificar posición horizontal y distancia inferior.
- Tiradores para modificar ancho y altura del fondo.
- Presets de composición.
- Deshacer y rehacer mediante botones, Ctrl+Z, Ctrl+Shift+Z y Ctrl+Y.
- El texto mantiene su tamaño, familia, peso y proporción.
- La conexión persistente de ProPresenter de Update 026 se conserva.

## Nota

El editor visual modifica la geometría real de 1920 × 1080. El zoom continúa
siendo únicamente una herramienta de visualización.
