# BMMS Update 026 — Foundation y motor de fondo

## Incluido

- Fondo con altura completa de 80 a 1080 px.
- Inicio superior independiente.
- Extensión inferior independiente.
- Opción `Extender el fondo hasta el borde inferior`.
- Classic Gradient y demás estilos utilizan las mismas variables.
- Preview con Ajustar, 50%, 75%, 100%, Ampliar y Pantalla completa.
- Configuración de ProPresenter persistida además en
  `data/bmms-connections.json`.
- El instalador incremental no incluye la carpeta `data`, por lo que no
  reemplaza la configuración del usuario.
- El tamaño y la proporción del texto no se modifican.
