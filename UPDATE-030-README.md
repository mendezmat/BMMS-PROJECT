# BMMS Update 030 — Smart Canvas Professional

## Editor visual

- Ocho manejadores profesionales.
- Centro de transformación visible.
- Selección vinculada al Scene Node `scripture-composition`.
- Inspector flotante X/Y/W/H durante drag y resize.
- Smart Guides para centro, Safe Area y guías persistentes.
- Safe Action y Safe Title independientes.
- Reglas horizontal y vertical.
- Doble clic sobre una regla para crear una guía.
- Doble clic sobre una guía para eliminarla.
- Cuadrícula y Snap separados: Grid, Centro y Safe Area.
- Ctrl/Cmd + rueda para zoom continuo.
- Espacio + arrastre para desplazar el lienzo.
- Doble clic en el lienzo para Ajustar.
- Ctrl/Cmd + 0 para Ajustar.
- Ctrl/Cmd + 1 para 100%.
- Navigator automático en zoom alto.

## Scene Engine

Las guías y restricciones del Smart Canvas quedan almacenadas en
`Composition.behavior`, dentro de la misma Scene utilizada por Scripture.
