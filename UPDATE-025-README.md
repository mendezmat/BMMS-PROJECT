# BMMS Update 025 — Escala geométrica

Esta versión parte de Update 024, que ya funciona correctamente.

## Cambios

- `Ancho de la composición` modifica:
  - ancho del panel;
  - espacios laterales;
  - sólidos;
  - fondos;
  - áreas seguras.

- `Altura de la composición` modifica:
  - altura del panel;
  - relleno vertical;
  - fondos;
  - degradados;
  - sólidos.

## El texto no cambia

La referencia, el versículo y la versión conservan:

- tamaño;
- proporción;
- ancho de caracteres;
- altura de caracteres;
- tipografía;
- peso;
- interlineado.

No se aplica `scale()` al texto ni al gráfico completo.
