# BMMS Update 024 — Escala X/Y de composición

Cambios:
- Nuevo control `Escala horizontal`.
- Nuevo control `Escala vertical`.
- Rango independiente de 50% a 200%.
- El escalado se aplica al gráfico completo, no solo al texto.
- `TAKE` ahora se muestra como `ENVIAR`.
- `CLEAR` ahora se muestra como `LIMPIAR`.
- `AUTO: OFF/ON` ahora se muestra como
  `AUTO: APAGADO/ENCENDIDO`.

No se modificaron las rutas de ProPresenter ni el flujo Manual.
